# Expense Tracker - Spring Boot Project

A full-stack expense tracker application built using Java Spring Boot, MySQL, and Thymeleaf.

## Features
- Add, update, delete expenses
- View all expenses
- Filter by category and date
- Total expense summary
- REST API + Web UI

## Tech Stack
- Java 17
- Spring Boot
- Spring Data JPA
- MySQL
- Thymeleaf
- Maven

## How to Run
1. Clone the repository
2. Configure MySQL in application.properties
3. Run as Spring Boot App
4. Open browser: http://localhost:8080/expenses
